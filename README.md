# título

Site que mapeia startups da região

* Abstrato
    * Tela única
    * Trazer nomes de startups
    * Botão para cadastrar startup (modal)
        * Nome da startup
        * Nome dos fundadores
        * Email
        * Site

### Devs

Dependencias: `yarn install` || `npm install`
Start: `npm start` || `yarn run start`
